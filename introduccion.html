Sistemas Algebraicos

  • Objetivo

    • El alumno asimilará la teoría de los sistemas algebraicos para luego en un futuro utilizar aquellos concepto de manera teórica-práctica

  • Introducción

    • En este tutorial se explicará lo que significa un sistema algebraico y después se dará varios ejemplos de sistemas algebraicos bien conocidos para tratar algunas de sus propiedades. Estos ejemplos muestran en diferentes sistemas algebraicos pueden tener varias propiedades en común. Esta observación proporciona una motivación para el estudio de sistemas algebraicos abstractos. Para dichos sistemas algebraicos, ciertas propiedades se consideran como axiomas del sistema. Cualquier resultado que es válido para un sistema abstracto es válido para todos aquellos para los cuales son verdad los axiomas.
